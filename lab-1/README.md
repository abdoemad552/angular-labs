# Lab 1

A simple ecommerce page with products list and a simple shopping cart.

This project is using Tailwind CSS for styling.

Applied concepts:
- `@Input`
- `@Output`
- Pipes (eg. `currency`)
- Injectable services

Current issues:
- Observable updates are not achieved yet. Will be applied in the next lab.
- Currently relying on `@Input` and `@Output` for data binding and updates. Next will use services.
- Not using signals yet.
